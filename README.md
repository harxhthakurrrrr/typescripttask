# Kanban Board Component

It implements a fully functional and reusable **Kanban Board component** using modern frontend technologies with clean architecture and scalable drag-and-drop logic.

---

## Objective

Create a Kanban board with three default columns:

- **Todo**
- **In Progress**
- **Done**

With support for:

- Add / Delete cards
- Drag & Drop between columns
- Preserve order within column
- Responsive layout
- Clean component structure

---

### Tech Stack

- **React (Functional Components)**
- **TypeScript**
- **Tailwind CSS**
- **@dnd-kit (Drag & Drop Library)**
- **Lucide Icons**
